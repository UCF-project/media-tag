Image plugin

Requirements

Based on Google's Shaka Player, Media-Tag needs a reference to this API.

You can add a <script src="[the dependency url]"></script> inside your page using Media-Tag or declare inside a configuration file the dependency url.

That can it do

It is used to render several type of images.

Media-Tag by default can regonizes these types:
  • image/png
  • image/jpg
  • image/jpeg
  • image/gif
  • image/svg+xml

And it can be used like this.

HTML

<media-tag # Some classes for the container class="slice centered justified" # Some classes for the render data-attr-class="fit-height" # Content source to process with Media-Tag src="/assets/videos/bbb_dash.mpd" # Data-type used to recognize the content type data-type="application/dash+xml" </media-tag>
