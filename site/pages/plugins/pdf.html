Image plugin

That can it do

It is used to render several type of images.

Media-Tag by default can regonizes these types:
  • image/png
  • image/jpg
  • image/jpeg
  • image/gif
  • image/svg+xml

And it can be used like this.

HTML

<media-tag # Some classes for the container class="slice centered justified" # Some classes for the render data-attr-class="fit-height" # Content source to process with Media-Tag src="/assets/images/media-tag-logo-text.png" # Data-type used to recognize the content type data-type="image/png" </media-tag>
