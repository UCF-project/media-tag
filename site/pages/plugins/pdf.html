Video plugin

That can it do

It is used to render several type of video.

Media-Tag by default can regonizes these types:
  • video/mp4
  • video/webm
  • video/ogg

And it can be used like this.

HTML

<media-tag # Some classes for the container class="slice centered justified" # Some classes for the render data-attr-class="fit-height" # Content source to process with Media-Tag src="/assets/videos/bbb.mp4" # Data-type used to recognize the content type data-type="video/mp4" </media-tag>
